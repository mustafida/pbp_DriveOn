\# DriveOn - Rental Mobil



\## Struktur

\- backend/ : Express + MySQL + Tripay

\- frontend/ : Flutter

\- database/ : driveon.sql



\## Cara jalanin backend

cd backend

npm install

copy .env.example ke .env lalu isi

npm run dev



\## Cara jalanin frontend

cd frontend

flutter pub get

flutter run
